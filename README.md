# Aplikasi Peminjaman Buku

Aplikasi **Peminjaman Buku** adalah aplikasi web sederhana berbasis **HTML, CSS, dan JavaScript** yang digunakan untuk mencatat data peminjaman dan pengembalian buku. Aplikasi ini mendukung operasi **CRUD (Create, Read, Update, Delete)** dan cocok digunakan sebagai **projek akhir / tugas kuliah**.

---

## Fitur

* Menambahkan data peminjaman buku
* Menampilkan daftar buku yang dipinjam
* Mengedit data peminjaman buku
* Menghapus data peminjaman buku
* Input tanggal peminjaman dan pengembalian
* Tampilan sederhana dan mudah digunakan

---

## Teknologi yang Digunakan

* **HTML** – Struktur halaman
* **CSS** – Tampilan dan desain antarmuka
* **JavaScript** – Logika CRUD dan manipulasi DOM

---

---

## Cara Menjalankan Aplikasi

1. Clone repository ini:

   ```bash
   git clone https://github.com/username/peminjaman-buku.git
   ```
2. Masuk ke folder project:

   ```bash
   cd peminjaman-buku
   ```
3. Buka file `index.html` menggunakan browser (Chrome, Edge, Firefox, dll)

---

## Cara Menggunakan

1. Masukkan **Judul Buku**
2. Pilih **Tanggal Peminjaman**
3. Pilih **Tanggal Pengembalian**
4. Klik tombol **Add** untuk menyimpan data
5. Gunakan tombol **Edit** untuk mengubah data
6. Gunakan tombol **Delete** untuk menghapus data

---

## Tujuan Proyek

* Menerapkan konsep CRUD menggunakan JavaScript
* Melatih manipulasi DOM
* Membuat aplikasi web sederhana tanpa framework
